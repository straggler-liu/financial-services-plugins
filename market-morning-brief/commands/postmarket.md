# 收盘复盘命令

立即触发收盘复盘分析，对比早盘建议与实际走势，总结今日市场规律，给出明日操作建议。

## 工作流程

### 第一步：解析参数

- `asia`（默认）：生成 A股+港股 收盘复盘
- 无参数：默认生成 A股+港股 复盘

### 第二步：执行复盘

**Docker 环境：**
```bash
docker-compose exec market-brief python src/main.py --now postmarket_asia
```

**本地 Python 环境：**
```bash
python src/main.py --now postmarket_asia
```

### 第三步：复盘内容

复盘报告包含：

1. **今日市场总结** - 三地指数涨跌复盘
2. **建议执行情况回顾** - 对比今日早盘建议与实际走势
   - 建议是否正确？准确率如何？
   - 哪些因素被低估/高估了？
   - 从第一性原则看，哪个变量（E/r/L/ρ）出现了偏差？
3. **今日关键事件复盘** - 各事件实际影响分析
4. **明日操作建议** - 优化后的策略
5. **明日重点关注标的** - 更新的自选清单

### 第四步：学习机制

系统会将今日建议和复盘结果保存到 `cache/last_strategy.json`，在明日分析时自动对比，形成闭环学习：

```
今日开盘前建议 → 实际走势验证 → 复盘偏差分析 → 优化明日策略
```

## 使用示例

```
/market-morning-brief:postmarket asia
/market-morning-brief:postmarket us
```

## 数据溯源说明

所有分析引用的数据均有明确来源：
- 行情数据：附带交易所/数据提供商名称
- 新闻事件：附带媒体名称和原文链接
- 经济数据：附带国家统计局/央行等官方来源
- 机构研报：附带研究机构名称和东方财富研报链接
