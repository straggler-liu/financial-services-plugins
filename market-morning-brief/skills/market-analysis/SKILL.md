# 市场分析技能 - 第一性原则框架

## 触发条件

当用户询问以下内容时自动调用此技能：
- 某只股票/板块今天应该怎么操作？
- 某个新闻/政策对市场有什么影响？
- 如何判断当前市场环境？
- 今天/明天的市场怎么看？

## 核心分析框架：第一性原则

### 市场价格决定方程

```
P = E / (r + ρ) × L

其中：
  P = 资产价格
  E = 未来盈利预期（Earnings Expectation）
  r = 无风险利率（Risk-free Rate）
  ρ = 风险溢价（Risk Premium，由情绪决定）
  L = 流动性乘数（Liquidity）
```

### 分析步骤（5步法）

**Step 1：识别事件类型**
```
政策类：央行/监管决定 → 主要影响 r 和 L
盈利类：财报/业绩预期 → 主要影响 E
宏观类：GDP/CPI/就业 → 影响 E 和 r
情绪类：外部冲击/黑天鹅 → 主要影响 ρ
```

**Step 2：确定影响变量**
```
对每个事件，必须明确：
  - 影响哪个变量（E/r/L/ρ）？
  - 影响方向（+正 / -负 / ~中性）？
  - 影响幅度（🔴高 / 🟡中 / 🟢低）？
  - 持续时间（短期<1周 / 中期1-3月 / 长期>3月）？
```

**Step 3：传导链条分析**
```
示例：
  美联储加息 → r上升 → 所有资产折现率提高 →
  高估值科技股首当其冲（P/E压缩） →
  成长股跑输价值股 →
  关注：银行（受益利差扩大）、公用事业（受损）
```

**Step 4：板块/个股映射**
```
政策利好/利空 → 直接受益/受损板块
利率变化 → 高/低负债公司，成长/价值股
流动性收紧/宽松 → 小盘/大盘，A股/港股资金流向
```

**Step 5：操作建议**
```
明确给出：
  - 方向（买入/减持/观望）
  - 时间维度（当天/本周/本月）
  - 价格区间（支撑位/阻力位）
  - 止损条件（什么情况下策略失效？）
```

## 三地市场特殊考量

### A股特性
- 散户占比高（~60%），情绪波动大
- 政策市特征：监管政策影响优先于基本面
- 北向资金是重要风向标（每日数据可查）
- 两市成交量是市场情绪的领先指标
- 涨跌停制度（±10%，科创板±20%）

### 港股特性
- 国际定价权（受美股和A股双重影响）
- 流动性溢价：在港上市中概股折价
- 南向资金（港股通）是重要支撑力量
- 恒生科技指数与纳斯达克高度相关
- 汇率风险（港元紧盯美元）

### 美股特性
- 机构主导（~80%），动量效应强
- 美联储政策是核心驱动变量
- 财报季（每季度一次）是重要催化剂
- 期权市场（VIX）是情绪晴雨表
- 标普500的市盈率水平影响全球风险偏好

## 数据来源规范

所有分析必须引用可溯源的数据，格式：

```
📊 [数据来源名称] 数据内容
   来源：URL（如有）
   时间：数据更新时间
```

可信数据来源优先级：
1. 🟢 官方权威：美联储官网、国家统计局、证监会、央行、港交所
2. 🟢 主流财经媒体：财联社、新华社、路透社、彭博
3. 🟡 财经数据平台：东方财富、新浪财经、Yahoo Finance
4. 🟡 量化数据库：AKShare、BaoStock（数据源于交易所）
5. 🔴 社交媒体：仅供情绪参考，不作为事实依据

## 风险提示模板

每次分析必须包含：

```
⚠️ 主要不确定性因素：
  1. [具体风险1]（概率：高/中/低）
  2. [具体风险2]

⚠️ 策略失效条件：
  若 [特定事件/数据] 出现，当前建议需重新评估

⚠️ 免责声明：
  本分析由AI生成，基于公开数据，仅供参考，
  不构成投资建议。投资有风险，决策需谨慎。
```
